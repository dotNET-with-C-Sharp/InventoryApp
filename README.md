# InventoryApp
A beginner friendly hands-on project for ASP.NET Core Web API using EF Core.

# Goal
The main goal is to strengthen backend development fundamentals and gain hands-on experience building scalable RESTful APIs using ASP.NET Core and Entity Framework Core.

---

# Tech Stack
| Technology | Tools |
|---|---|
| ASP.NET Core Web API | Visual Studio Community 2026 |
| C# | SQL Server 2025 Express |
| Entity Framework Core | SQL Server Management Studio |
| SQL Server | Postman |
| Swagger UI |  |

---

# Key Learnings
## ASP.NET Core Web API Architecture
- Understanding the end-to-end Web API request flow
- Working with:
  - Models
  - Controllers
  - Repository/Data Access Layer

---

## CRUD Operations with In-Memory Database
- Implemented Create, Read, Update, and Delete operations
- Used an in-memory database for initial development and testing

---

## API Testing with Swagger UI (also Postman)
- Using Swagger UI to:
  - Explore API endpoints
  - Test HTTP methods
  - Validate and Send request payloads
  - Understanding RESTful API conventions and status codes

---

## SQL Database Integration using Entity Framework Core
- SQL Server Express and SSMS Setup
- Created and Connected the application to a SQL database
- Performed CRUD operations with multiple entities
- Configured shared DbContext

---

## EF Core Migrations
- Added new columns and updated entity models
- Generated and applied database migrations

---

## Entity Relationships in EF Core
- Implemented relationship
- Configured:
  - Foreign Keys
  - Navigation Properties
  - One-to-Many relationships

## Additional Concepts Explored
- DTOs (Data Transfer Objects)
- Model Validation using Data Annotations
- Nullable vs Required properties
- JSON serialization behavior, Circular Reference
- API response handling and error management
